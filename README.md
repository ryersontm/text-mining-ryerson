# Final Projects for Text Mining Course, DS 8008, at Ryerson University, Toronto

This repoistory hosts the final assignments (projects) for text mining course at Ryerson University. These projects are mostly replication studies of research papers in the field of Natural Language Processing.

This repository is organized by academic years, browse through each folders above to see the interesting projects.
